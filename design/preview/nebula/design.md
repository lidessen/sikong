## Nebula — Design Direction

**Theme**: Bold Colorful Product Landing  
**Color Palette**: Deep navy (#0a0f1e) base, violet (#7c3aed) primary with amber (#f59e0b) and pink (#ec4899) accents — a brand-forward editorial palette  
**Key Visual Elements**: Massive typography (up to 5.5rem), 3-color gradient text (violet→amber→pink), glowing multi-source radial backgrounds, "eyebrow" badge, colored step numbers

**Layout**:
1. Sticky nav with gradient brand text
2. Hero: eyebrow badge, two-line title (second line in 3-color gradient), tagline, CTA buttons, 4 meta badges with colored dots
3. "How It Works" — 4-column horizontal stepper with numbered steps and gradient connector line
4. Features grid (6 cards, each with colored icon-background box)
5. Runtime Badges — horizontal row of 4 compatibility badges
6. Architecture — 3-block flow with triangle arrow separators
7. Comparison — HTML table with alternating row backgrounds
8. Install — 2-column grid code blocks (Curl, npm)
9. CTA section with trust/community indicators row
10. Footer — 3-column grid (Brand, Product, Connect)

**Design Principles**: Editorial polish, brand memorability, rich visual hierarchy, motion (pulse/float keyframes), platform-scale feel
