## Architect — Design Direction

**Theme**: Terminal-Inspired Developer Landing  
**Color Palette**: Deep navy (#0b1120) base, blue (#3b82f6) accent with cyan (#06b6d4) and purple (#a78bfa) gradient text  
**Key Visual Elements**: Terminal window chrome in hero with red/yellow/green dots and simulated CLI output, 64px grid background pattern, monospace accents throughout

**Layout**:
1. Sticky nav with bracket-branding `[sikong]`
2. Full-viewport hero: gradient heading, tagline, CTA buttons, terminal window widget
3. Stats bar (4 runtimes, 1.2M+ tokens, 100% open source, MIT)
4. Features grid (8 cards with glowing accent lines)
5. Architecture diagram — 3 layers connected by arrows, middle layer highlighted
6. Comparison — side-by-side "Without Sikong" (red) vs "With Sikong" (green)
7. Install — tabbed code blocks (Curl/NPM) with file-headers
8. CTA section
9. Footer — 4-column grid (Brand, Product, Community, Resources)

**Design Principles**: Developer-authentic, CLI-native feel, technical credibility, information-dense but well-structured
