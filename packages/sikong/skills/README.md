# sikong skills

Agent skills for the [open agent-skills ecosystem](https://skills.sh/) (`npx skills`),
installable into Claude Code, Codex, Cursor, and others.

| Skill | What it gives an agent |
| --- | --- |
| [`sikong`](sikong/SKILL.md) | How to drive sikong well — workflows, grounded + lead-authored acceptance gates, auto-mode workers, and the common pitfalls. |

Install:

```sh
npx skills add lidessen/sikong --skill sikong
```
